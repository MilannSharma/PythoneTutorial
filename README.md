# Python Course

## Description

This repository is a comprehensive collection of Python tutorials, code examples, and resources designed to help learners progress from basic to advanced Python programming. It includes organized chapters, practice sets, object-oriented programming concepts, and utility scripts for generating PDF documentation from text files.

## Features

- **Tutorials and Code Examples**: Step-by-step guides and code snippets for various Python topics.
- **PDF Generation**: Scripts to convert text files into PDF tutorials using ReportLab.
- **Organized Structure**: Folders for basic, advanced, and specialized topics.
- **Practice Sets**: Hands-on problems and solutions for reinforcement.

## Folder Structure

- **Basic Python/**: Fundamental Python concepts, including comments, variables, data types, strings, tuples, dictionaries, lists, sets, conditionals, operators, loops, functions, recursion, and I/O files.
- **Advanced Python/**: Advanced topics such as classes, class methods, constructors, exception handling, inheritance, operator overloading, property decorators, and more.
- **Oops concepts/**: Object-oriented programming concepts with examples and practice sets.
- **Chapters/**: Detailed chapters on Python topics, including rough work and examples.
- **moduls/**: Custom modules and libraries.
- **Practice sets/**: Practice problems organized by chapters.
- **Scripts**:
  - `convert_txt_to_pdf.py`: Converts .txt files in specified folders to PDFs.
  - Other utility scripts for documentation and file management.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MilannSharma/Pythone-Tutorial-.-py.git
   cd Python-Cource
   ```

2. Install required dependencies:
   ```bash
   pip install reportlab
   ```

## Usage

- Explore the code examples in the respective folders.
- To generate PDFs from text files, run:
  ```bash
  python convert_txt_to_pdf.py
  ```
  This will create PDFs in the 'Basic Python' and 'Advanced Python' folders.

- Run individual Python scripts to test concepts, e.g.:
  ```bash
  python Chapters/1Comments.py
  ```
